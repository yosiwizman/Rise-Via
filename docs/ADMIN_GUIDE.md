## Product Management

### Adding Products
1. Go to **Products** section
2. Click **Add Product**
3. Fill out product information:
   - Name, price, category
   - THC/THCA percentages
   - Strain type (Sativa/Indica/Hybrid)
   - Effects and description
   - Images and inventory levels

### Product Fields
- **Name**: Product display name
- **Price**: Current selling price
- **Category**: Product category (flower, edibles, concentrates)
- **THCA %**: Potency percentage
- **Strain Type**: Sativa, Indica, or Hybrid
- **Effects**: Array of expected effects
- **Description**: Detailed product description
- **Images**: Product photos (multiple supported)
- **Inventory**: Current stock level
- **Active**: Product visibility status

### Editing Products
1. Search for product in product list
2. Click **Edit** button
3. Update any field inline or use full editor
4. Changes save automatically

### Bulk Operations
1. Select multiple products using checkboxes
2. Choose bulk action:
   - **Activate/Deactivate**: Change product status
   - **Adjust Prices**: Apply price changes
   - **Delete**: Remove products

#### Price Adjustment Examples
- `+5`: Add $5 to each product
- `-10`: Subtract $10 from each product
- `*1.1`: Increase prices by 10%

### Inventory Management
- Monitor stock levels in real-time
- Set low stock alerts
- Update inventory quantities
- Track product performance

### Product Import/Export
- **Export CSV**: Download product catalog
- **Import CSV**: Bulk upload products
- **Image Upload**: Batch image processing

### Order Statuses
